## 🚀 Featured Projects

### 🔍 SQLi_checker
**Advanced SQL Injection Vulnerability Scanner**

A sophisticated, multi-threaded SQL Injection vulnerability scanner designed for security researchers and penetration testers. This tool automates the process of discovering SQLi vulnerabilities across multiple attack vectors.

- **Repository:** [hoxedzik666/SQLi_checker](https://github.com/hoxedzik666/SQLi_checker)
- **Key Features:**
  - Multi-threaded scanning for improved performance
  - Support for multiple attack vectors:
    - URL parameters
    - HTTP headers
    - Cookies
    - POST data
  - Comprehensive vulnerability reporting
  - Easy integration into security workflows
- **Status:** 🟢 Active Development
- **License:** Open Source
- **Use Cases:** Penetration testing, security auditing, vulnerability assessment

### 📱 TelegramGroupsAndChannelsInspector
**Telegram Communities Analytics & Inspection Tool**

A Python-based intelligence gathering tool for analyzing Telegram groups and channels. Perfect for security researchers, community moderators, and social network analysts.

- **Repository:** [hoxedzik666/TelegramGroupsAndChannelsInspector-main](https://github.com/hoxedzik666/TelegramGroupsAndChannelsInspector-main)
- **Key Features:**
  - Comprehensive group/channel metadata extraction
  - Community analysis capabilities
  - Data export functionality
  - User-friendly interface
  - MIT License (Open Source)
- **Status:** 🟢 Active Development
- **Language:** Python
- **Use Cases:** Community analysis, threat intelligence, research, moderation assistance
